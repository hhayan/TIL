- 파일 설명
1. CSV (Comma Separated Values)
정의: 쉼표(,)로 구분된 평문 텍스트 형식의 데이터 파일
구조: 테이블(엑셀과 비슷한 형태)을 텍스트로 표현
2. JSON (JavaScript Object Notation)
정의: 데이터를 구조화하여 표현하는 데 사용하는 형식, JavaScript에서 유래했지만 다양한 언어에서 사용됨
구조: 객체({}), 배열([]), 키-값 쌍으로 구성됨

###함수

DISTINCT: 중복 제거 <- 오타주의
POWER(10, 6): 10의 6승
